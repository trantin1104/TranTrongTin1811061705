# ShoeStore - Trang Web Bán Giày

## Giới Thiệu
ShoeStore là một trang web bán giày được phát triển bằng C# ASP.NET MVC. Dự án cung cấp các chức năng quan trọng như xem sản phẩm, quản lý danh mục, giỏ hàng, đăng nhập, đăng xuất và phân quyền người dùng.

## Chức Năng
### 1. Quản Lý Sản Phẩm
- Xem danh sách sản phẩm
- Thêm mới sản phẩm
- Chỉnh sửa thông tin sản phẩm
- Xóa sản phẩm

### 2. Quản Lý Danh Mục
- Xem danh sách danh mục
- Thêm danh mục mới
- Chỉnh sửa danh mục
- Xóa danh mục

### 3. Quản Lý Giỏ Hàng
- Thêm sản phẩm vào giỏ hàng
- Xem danh sách sản phẩm trong giỏ hàng
- Xóa sản phẩm khỏi giỏ hàng
- Cập nhật số lượng sản phẩm

### 4. Quản Lý Tài Khoản
- Đăng nhập, đăng xuất
- Phân quyền người dùng (Admin, User)
